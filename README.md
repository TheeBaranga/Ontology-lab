# Ontology-lab

## Description
This project represents a simple ontology for a university academic system using Python. It uses classes, objects, and dictionaries to build a knowledge base that tracks student enrollments, faculty teaching assignments, department mapping, and prerequisite reasoning for courses like APT3020 and CMS3700.

## Ontology Concepts
- Students
- Lecturers
- Courses
- Departments
- Classrooms

## Relationships
- Student enrolled in Course
- Lecturer teaches Course
- Course belongs to Department
- Course has prerequisite
- Course taught in Classroom

## How to Run
Ensure you have Python installed on your system. Navigate to the project directory in your terminal and execute:
```bash
python ontology_lab.py
